# django-oidc
OpenID connect server based on Django


### How to run ?

- setup your environment

```
./setup.sh
```

- setup and run the oidc provider server

```
./django_oidc/setup.sh
./django_oidc/run.sh
```

- run the relying party client

```
./rp_client/run.sh
```
